Modelo de Negocio

La Asociación de Agricultores de Palto en Huamanpali - Alto Larán es una organización dedicada a promover el desarrollo agrícola de sus asociados, fomentando la producción y comercialización de palta de calidad, como parte de su proceso de modernización y difusión de información, se plantea el desarrollo de un sitio web informativo que permita fortalecer la comunicación con agricultores, clientes potenciales, instituciones y público en general.

El presente proyecto tiene como finalidad analizar, diseñar e implementar una base de datos relacional que soporte las funcionalidades del sitio web de la asociación. La base de datos permitirá gestionar la información generada a través del formulario de contacto y del libro de reclamaciones, así como organizar los datos necesarios para el correcto funcionamiento del sistema.

Para el desarrollo del proyecto se aplicarán las etapas de análisis de requisitos, diseño conceptual, diseño lógico, diseño físico e implementación en MySQL, siguiendo las buenas prácticas de diseño de bases de datos. Asimismo, se elaborarán los scripts de creación de la estructura de la base de datos, carga de datos maestros, carga de datos transaccionales y consultas necesarias para demostrar el funcionamiento del sistema.
